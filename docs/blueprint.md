# **App Name**: SkillConnect

## Core Features:

- Landing Page: High-end animated hero section with clear calls to action.
- Courses Listing: Display available courses in a grid layout with search and filter options.
- Course Details: Show detailed information about a specific course, including curriculum and instructor.
- Communities Listing: Display available communities with filtering options.
- Community Details: Show community details with a feed of posts and community information.

## Style Guidelines:

- Primary color: Dark blue (#3F51B5) to convey trust and knowledge.
- Background color: Very light gray (#F5F5F5) for a clean and modern look.
- Accent color: A vibrant orange (#FF9800) for call-to-action buttons and highlights.
- Body and headline font: 'Inter', sans-serif, for a modern and readable feel.
- Use minimalist icons from a library like Feather icons, focused on clarity and simplicity.
- Employ a clean and spacious layout with a clear hierarchy, utilizing shadcn/ui components to maintain consistency.
- Utilize Framer Motion for subtle animations like fade-in effects on scroll, hover transitions, and micro-interactions on buttons.
